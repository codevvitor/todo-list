# 📝 To-Do List

Aplicação de lista de tarefas com persistência de dados via LocalStorage.

## 🔗 Demo ao vivo
[Clique aqui para ver](https://codevvitor.github.io/todo-list)

## ✨ Funcionalidades
- Adicionar e remover tarefas
- Marcar tarefas como concluídas
- Dados salvos no navegador (LocalStorage)
- Interface responsiva

## 🛠️ Tecnologias
HTML5 · CSS3 · JavaScript Vanilla
